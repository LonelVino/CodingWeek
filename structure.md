# Structure of This Project

## Main Files

- [gui.py](/gui.py): Definition for all of the pages and of the graphical user interface

## Assets Folder
- [./assets/](/assets/): Images and gifs for the gui

## Support Files
- [ant.py](/ant.py): Implements the class Ant
- [cell.py](/cell.py): Implements the class Cell
- [gui_utils.py](/gui_utils.py): Implements the classes BFControlId, Button, Label, InputBox, CellButton Rectangle which are used as widgets
- [map.py](/map.py): Implements the class Map
